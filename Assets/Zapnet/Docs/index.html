Demo Project

The demo project included with Zapnet demonstrates how to a simple Zapnet project is setup. The Initializer behaviour will handle the hosting of the server or the connection to the server. It provides an example of how IServerHandler and IClientHandler are implemented to allow for authentication and performing other useful functions such as sending and receiving initial data to a newly connected player.

Network Hitboxes

The demo project demonstrates how network hitboxes work with synchronized projectiles being instantiated when a player fires their weapon. Hitboxes are rewound to the position the server believes they were at when the client sent the fire event.

Events

The demo project demonstrates how network events work. The RegisterPacket<T> method must be called in the same order on the server and client.
